# M7 平台化增强阶段 — 开发进度

> 时间范围：T49 ~ T60 | 状态：全部完成

## 已完成任务

### T49：JWT 认证体系（PyJWT + 双Token + Refresh）

**交付物：**
- `app/core/security.py` — JWT 工具（access_token 30min + refresh_token 7d）
- `app/api/users.py` — 登录/注册/刷新/登出/改密 API
- `app/models/user.py` — User、Team、RefreshToken 模型
- 密码 bcrypt 哈希，Token 黑名单机制

**API 端点：**
- `POST /api/auth/login` — 登录获取双 Token
- `POST /api/auth/refresh` — 刷新 Access Token
- `POST /api/auth/logout` / `logout-all` — 登出
- `POST /api/auth/change-password` — 修改密码
- `GET /api/auth/me` — 当前用户信息

---

### T50：数据权限隔离（RBAC + team_id）

**交付物：**
- `app/core/permissions.py` — `team_filter()` 查询过滤 + `check_resource_access()` 资源鉴权
- 所有业务 API（cases/executions/environments/mocks/benchmarks/pipelines）已接入 `get_current_user` + `team_filter`
- admin 角色可跨团队访问，普通用户只能看到本团队数据
- 用户管理 API 限 admin 角色

**权限模型：**
- admin：全局管理，跨团队
- member：团队内 CRUD
- viewer：只读

### T54：WebSocket 实时推送

**交付物：**
- `app/api/ws.py` — WebSocket 端点 + ConnectionManager（团队隔离广播）
- `composables/useWebSocket.ts` — 前端 WebSocket composable（自动重连 + 心跳）
- 执行创建/完成时自动推送 `execution:created` / `execution:update` 事件

**协议：**
- 连接：`ws://host/ws?token=<access_token>`
- 心跳：客户端发 `ping`，服务端回 `{"event":"pong"}`
- 消息格式：`{"event": "xxx", "data": {...}}`

### T58：gRPC 协议完整支持

**交付物：**
- `mwjrunner/protocols/grpc.py` — 完整 gRPC 适配器

**能力：**
- Unary RPC 调用
- Server Streaming RPC
- 动态 Server Reflection（无需预编译 stub）
- 通用 JSON 序列化调用（fallback）
- TLS / 非 TLS 连接
- Metadata 传递
- 超时控制

**依赖：** `grpcio`, `grpcio-reflection`, `protobuf`（可选安装）
