# MwjRunner 使用手册

本手册已拆分为两部分，方便不同阶段的用户快速定位内容。

## 入门篇

适合刚接触 MwjRunner 的用户。涵盖安装准备、示例服务、第一个用例、执行命令、断言基础、变量提取、Console/JSON 报告等内容。

→ [使用手册-入门篇](使用手册-入门篇.md)

## 进阶篇

适合已掌握基础的用户。涵盖 HTML 报告、退出码、敏感信息脱敏、数据驱动、并发执行、hooks、Postman/OpenAPI 导入、通知、认证、协议扩展（WebSocket/gRPC）、质量门禁、报告插件、Web 管理平台等高级功能。

→ [使用手册-进阶篇](使用手册-进阶篇.md)
