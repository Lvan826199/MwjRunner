# 开发问题跟踪

本文档用于记录 MwjRunner 开发过程中发现但未在当前任务内立即修复的问题，便于后续按优先级集中处理。

## 优先级定义

- P0：阻塞主流程、导致核心功能不可用、存在严重安全风险的问题。
- P1：影响核心能力正确性、稳定性或安全边界，需要尽快修复的问题。
- P2：影响可维护性、诊断体验或部分边界场景的问题。
- P3：建议项、文档清理、体验优化或后续增强。
